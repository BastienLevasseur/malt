Global statistics

Metric Value Descr
Peak virtual memory {{globalStats.virtualMem}} Peak of total virtual memory obtained by tracking /proc/self/statm. It contain all allocated objects, global variables and binary code.
Peak physical memory {{globalStats.physicalMem}} Peak of total physical memory obtained by tracking /proc/self/statm. It contain all allocated objects, global variables and binary code.
Peak requested memory {{globalStats.requestedMem}} Peak of requested memory managed by malloc, free... It contain all dynamically allocated objects.
Cumulated memory allocations {{globalStats.cumulAllocs}} Sum of all dyanmic memory allocation throuth malloc, calloc, realloc....
Recycling ratio {{globalStats.recyclingRatio}} Ratio of cumulated memory allocations over maximum requested memory. Ideally it might be 1 if the program allocate the memory on start and free on end.
Allocation count {{globalStats.allocCount}} Total number of all dyanmic memory allocation with malloc, calloc, realloc....
Smallest allocations {{globalStats.smallestAlloc}} The smallest request size received by malloc, calloc...
Largest allocations {{globalStats.largestAlloc}} The largest request size received by malloc, calloc...
Largest stack {{globalStats.largestStack}} Largest stack used by the program during the execution of all threads. Obtained by tracking RSP register while entering and exiting functions.
